[Homebrew](https://brew.sh/) ist ein Paketmanager für macOS, [Chocolatey](https://chocolatey.org/) für Windows. Beide ermöglichen dir die Installation von Softwarepaketen und Bibliotheken über die Kommandozeile.

Wozu soll das gut sein, fragst du dich? Ganz einfach: Mit Homebrew und Chocolatey kannst du Softwarepakete und Bibliotheken mit einem Befehl installieren, ohne dass du erst bei Google nach dem Download-Link suchst, die Datei herunterlädst, doppelklickst und durch den Installationsprozess klickst, nur um anschließend die Installationsdatei wieder zu löschen.

Die beiden Paketmanager sparren dir also viel Zeit. Und so insstallierst du sie:

## Homebrew

Unter macOS kannst du Homebrew mit dem folgenden Befehl installieren:

```shell
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Überprüfe die Installation von Homebrew mit dem folgenden Befehl:

```shell
brew --version
```

Das sollte dir die aktuelle Version von Homebrew anzeigen.

### Software installieren

Um dir zum Beispiel Node.js zu installieren, kannst du den folgenden Befehl verwenden:

```shell
brew install node
```

Oder du installierst dir Firefox mit dem folgenden Befehl:

```shell
brew install firefox
```

### Software aktualisieren

Das Update der von dir installierten Software erfolgt mit dem folgenden Befehl:

```shell
# Installation aller Updates
brew upgrade

# Installation eines spezifischen Updates
brew upgrade <paketname>
```

Homebrew selbst kannst du mit dem folgenden Befehl aktualisieren:

```shell
brew doctor && brew update
```

Einfach, oder? Weitere Informationen zu Homebrew findest du auf der [offiziellen Website](https://brew.sh/).

## Chocolatey

Unter Windows hast du Chocolatey bereits installiert, wenn du Node.js nach unserer [Anleitung](/guides/setup#nodejs) installiert hast. Andernfalls kannst du Chocolatey mit dem folgenden Befehl installieren:

```shell
Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))
```

Überprüfe die Installation von Chocolatey mit dem folgenden Befehl:

```shell
choco --version
```

Das sollte dir die aktuelle Version von Chocolatey anzeigen.

### Software installieren

Um dir zum Beispiel Node.js zu installieren, kannst du den folgenden Befehl verwenden:

```shell
choco install nodejs
```

Oder du installierst dir Firefox mit dem folgenden Befehl:

```shell
choco install firefox
```

### Software aktualisieren

Das Update der von dir installierten Software erfolgt mit dem folgenden Befehl:

```shell
choco upgrade all
```

Chocolatey selbst kannst du mit dem folgenden Befehl aktualisieren:

```shell
choco upgrade chocolatey
```

Einfach, oder? Weitere Informationen zu Chocolatey findest du auf der [offiziellen Website](https://chocolatey.org/).
